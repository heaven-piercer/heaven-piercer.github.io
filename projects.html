ULTRAHUD

A UI modification for TF2 which mimics the aesthetic of the one found in ULTRAKILL. ULTRAHUD has many new features, and has been a project I have been working on for over a year now. It is built off of the popular TF2 HUD rayshud.

Features:

  • Redesigned HUD
  • Minmode for visual clarity
  • New charge meters
  • Completely overhauled Menus
  • Extensive color customizations
  • New fonts
  • 62 new crosshairs and hitmarkers

Resources/Tools I Use:

  • Visual Studio Code
  • TF2 Hud Reference (Jarateking)
  • Pixlr E
  • Dafont
  • Github Desktop
  • VTFEdit
  • VSCode VDF Extension (Revan)

Cookie Counter

A recreation of the game Cookie Clicker I made with a few friends for a school project.

The game is coded in Javascript using CodeHS.

Features:

  • 10 Buildings
  • Fully reworked (and busted) cursor
  • Randomly generated cookies
  • Infinite upgrades
  • Customizable Background/Font
  • Background Music by Hakita

The code is open source on GitHub.
