Cookie Clicker

I have (with the help of a few friends) recreated the game Cookie Clicker for a school project.

The game is coded in Javascript using CodeHS.

Features:

  • 10 Buildings
  • Fully reworked (and busted) cursor
  • Randomly generated cookies
  • Infinite upgrades
  • Customizable Background/Font
  • Background Music by Hakita
